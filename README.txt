سكولار — نظام إدارة ومتابعة الطلاب للمدارس
Scholar — School Management & Student Follow-up System
=======================================================

نموذج تفاعلي (Mockup) — جاهز للعرض.

كيفية الفتح:
1) فُكّ ضغط الملف (Unzip).
2) افتح ملف  login.html  بأي متصفح حديث.
3) من صفحة الدخول اختر نوع الحساب: مدير / معلم / طالب — ثم اضغط الدخول.

ثلاث واجهات (Role switcher في صفحة الدخول):
- مدير  → index.html          (لوحة إدارة المدرسة الكاملة)
- معلم  → teacher.html        (رصد الدرجات، الحضور، صفوفي وجدولي، الواجبات والاختبارات)
- طالب  → student-portal.html (درجاتي، جدولي وحضوري، واجباتي واختباراتي، الإعلانات)

صفحات أخرى:
- landing.html  : الصفحة التعريفية / التسويقية
- student.html  : ملف الطالب التفصيلي (يفتحه المدير من قائمة الطلاب)
- app.js        : طبقة التفاعل (لا يُفتح مباشرة)

ملاحظة: نموذج للعرض فقط — لا توجد قاعدة بيانات حقيقية؛ الأزرار تُظهر نوافذ وإشعارات تجريبية.
